- Slash Command: `/kori`

## What KoriUI Includes
- Rebuilt Command Deck configuration shell.
- Unit frames and castbars.
- Action bar enhancements.
- Minimap and datatext modules.
- Cooldown manager and class/resource tracking.
- Mythic+ tools and utility modules.
- Automation and quality-of-life helpers.
- Brewmaster stagger bar module.

## Command Deck Highlights
- New team-owned branding and metadata.
- Header shows current addon version.
- Header includes official Discord invite link.
- Drag the window by its top header.
- Resize from the bottom-right corner.
- Panel size persists between reloads.

## Slash Commands
- `/kori` open Command Deck.
- `/koriui` alias for `/kori`.
- `/rl` safe reload flow.
- `/kb` quick keybind mode.
- `/cdm` cooldown settings toggle.
- `/stagger` stagger module commands.

## Installation
1. Download the latest release zip.
2. Extract the `KoriUI` folder.
3. Place `KoriUI` in:
   - `World of Warcraft/_retail_/Interface/AddOns/`
4. Launch the game and enable KoriUI.
5. Type `/kori` in chat.

## CurseForge Release Packaging
Release zip must contain:
- Top-level folder: `KoriUI`
- Inside folder: `.toc`, `load.xml`, `utils/`, `libs/`, `assets/`, and other addon files.

Example release filename:
- `KoriUI-3.0.2-curseforge.zip`
